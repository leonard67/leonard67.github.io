## Python Coding for Accounting and Finance Research

This workshop is part of the **AFAANZ Doctoral Education Network (AFDEN)** program, designed for doctoral students and early-career researchers in accounting and finance who want to leverage Python for empirical research.

The workshop covers practical Python programming skills applied to accounting and finance research contexts, including data collection, processing, and empirical analysis.

This workshop is provided in September every year. To register, see the information in [AFDEN website](https://www.afaanz.org/doctoral-education).

All the course materials are shared in the [GitHub repository](https://github.com/leonard67/AFDEN-Python-Workshop).

### Topics Covered

- Python fundamentals for data analysis
- Data collection and web scraping for financial data
- Data cleaning and manipulation with pandas
- Empirical research workflows
- Text analysis and natural language processing for accounting research
- Statistical analysis and visualization

### Target Audience

Doctoral students and researchers in accounting, finance, and related disciplines with little to no prior programming experience.
