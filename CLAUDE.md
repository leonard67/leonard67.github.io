# CLAUDE.md

## Overview

This is a personal academic website for Leonard Leye Li (Senior Lecturer, UNSW Business School), hosted on GitHub Pages.

## Structure

The personal website contains the following tabs:

- Home/contact page
- Publications and working papers
- Courses and supervision
- Python workshop content

## Deployment

Push to the `main` branch; GitHub Pages serves the site automatically. No build step required.

## Conventions

**Page structure:** Every page shares an identical `<header>` with nav links. The active page gets `class="active"` on its nav `<a>`. The `<footer>` on the home page includes "Updated: [month year]"; other pages omit the date.
