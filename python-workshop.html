Python for Accounting and Finance Research

This workshop introduces Python tools for empirical research in accounting and finance, with a focus on reproducible workflows and working with large financial databases.

Intended audience

HDR students, early-career researchers, and practitioners interested in:

  • Automating data collection and cleaning
  • Working with CRSP/Compustat-style panel data
  • Implementing empirical models used in capital markets research
  • Documenting and version-controlling research code (Git/GitHub)

Indicative outline

  1. Introduction to Python and Jupyter notebooks
  2. Data structures in pandas and numpy
  3. Merging, reshaping, and cleaning financial statement data
  4. Event-study style calculations and portfolio sorts
  5. Basic visualization and regression analysis
  6. Reproducible research: environments, scripts, and GitHub
